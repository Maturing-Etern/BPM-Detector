# BPM检测器 v2.0

一个简单易用的音频BPM检测工具，支持全局BPM和多段滑动窗口分析。

## 功能特点

- 🎵 **全局BPM检测** - 计算整首歌曲的速度
- 📊 **多段分析** - 滑动窗口显示BPM变化趋势
- 🎯 **节拍显示** - 以 [小节,拍数,分母] 格式显示
- 🖥️ **图形界面** - 简单直观，无需命令行
- 📦 **单文件打包** - 无需Python环境，开箱即用
