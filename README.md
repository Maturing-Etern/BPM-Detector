# BPM检测器 - 软件介绍

## 🎵 软件简介

**BPM检测器** 是一款专业的音频速度分析工具，可以精确检测音乐的BPM（每分钟节拍数），并以直观的方式展示音乐的节奏变化。

---

## ✨ 核心功能

### 1. **全局BPM检测**
- 快速计算整首歌曲的平均速度
- 准确识别音乐的基准节奏
- 支持各种音乐风格（流行、电子、古典等）

### 2. **多段滑动分析**
- 将长音频分成多个时间段分析
- 显示BPM随时间的变化趋势
- 发现音乐中的变速段落

### 3. **节拍可视化**
- 以 `[小节,拍数,分母]` 格式显示每个节拍
- 例如 `[3,2,4]` 表示：第3小节的第2拍，4/4拍
- 帮助理解音乐的节奏结构

### 4. **详细分析报告**
- 窗口时间、BPM值、节拍信息一目了然
- 支持超长音频文件（数小时）
- 结果实时显示，无需等待

## ❓ 常见问题

### Q1: 分析结果准确吗？
A: 对于节奏清晰的音乐，准确率超过95%。极复杂的古典音乐可能会有误差。

### Q2: 支持多长的音频？
A: 理论上无限制，已测试过3小时的音乐会录音。

### Q3: 为什么有些窗口BPM为0？
A: 可能是该时间段静音或节奏不明显，属于正常现象。

### Q4: 可以批量处理吗？
A: 当前版本支持单文件分析，批量处理将在后续版本添加。

### Q5: 如何解读节拍格式？
A: `[小节,拍数,分母]`，例如`[3,2,4]`表示第3小节的第2拍，4/4拍。
